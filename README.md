# DieScore

Wordle-inspired mobile-friendly dark-mode dice strategy game for 3–6 players (human and/or computer AI).

## Run

Open `index.html` in a browser.

## Game summary

- 3 to 6 players take turns on the same device.
- Each player has one die per color from the selected color set (standard set: red, yellow, black, white, gray, blue).
- On each turn, choose two cards from a row of four and cross-apply color/action between those cards.
- Actions: Slide Right, Slide Left, Flip, Reroll, +2/-2.
- After each turn, trophy is awarded by score rank:
  - Gold (1st): 3 pts
  - Silver (2nd): 2 pts
  - Bronze (3rd): 1 pt
- Deck can be Standard (60 cards) or Double (120 cards for longer games).
